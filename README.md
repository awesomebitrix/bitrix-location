Сильно с таблицами еще не разбирался, при беглом просмотре пришел к такому заключению:

### Таблицы классификатора адресов: ###
- **b_sale_loc_type** типы объектов классификатора
- **b_sale_loc_type_name** наименования типов объектов классификатора
- **b_sale_loc_ext** Почтовые индексы и коды Яндекс.Маркет
- **b_sale_loc_ext_srv** типы индексов из таблицы b_sale_loc_ext (столбец SERVICE_ID [1 => 'Коды Яндекс.Маркет', 2 => 'Почтовый индекс') 
- **b_sale_loc_name** таблица с областями, городами, районами, улицами		
- **b_sale_location** промежуточная таблица, для соединения областей с городами, городов с улицами и т.д.


### Таблицы для подбора по фразам ###
- **b_sale_loc_search_world** для поиска по фразам в строке подбора метосположения
- **b_sale_loc_search_w2l** промежуточная таблица для соединения фраз из b_sale_search_world с b_sale_location
- **b_sale_loc_search_chain** еще какая-то промежуточная таблица )))
